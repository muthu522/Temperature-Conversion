# Temperature-Conversion
A simple Temperature Converter web app built using HTML, CSS, and JavaScript. It allows users to convert values between Celsius and Fahrenheit with an easy-to-use interface and real-time results.

A simple web-based temperature conversion tool that allows users to convert temperatures between Celsius and Fahrenheit.

##  Features
- Converts **Celsius → Fahrenheit**
- Converts **Fahrenheit → Celsius**
- Simple and responsive UI
- Built using HTML, CSS, and JavaScript

##  Technologies Used
- HTML5
- CSS3
- JavaScript 

##  How to Use
1. Enter the temperature value in the input box.
2. Select the conversion type:
   - Celsius to Fahrenheit
   - Fahrenheit to Celsius
3. Click the **Submit** button to see the result.
